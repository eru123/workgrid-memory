## Maintainer Responsibilities

Maintainers are responsible for:

- Reviewing pull requests
- Managing releases
- Triaging issues
- Handling vulnerability reports
- Enforcing the code of conduct
- Protecting user privacy and project security boundaries

## Security Responsibility

Maintainers should treat profile memory, workspace indexes, local files, and MCP access as sensitive areas.

Security issues should be handled according to `SECURITY.md`.
