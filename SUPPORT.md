# Support

This file explains where to get help with WorkGrid Memory.

## Project Status

WorkGrid Memory is currently in planning and early development.

Expect breaking changes, incomplete features, and occasional architectural reconsideration. This is normal. Software begins as a dream and then becomes a backlog with receipts.

## Where to Ask Questions

Use GitHub Issues for:

- Bug reports
- Feature requests
- Documentation problems
- Reproducible errors
- Security-safe technical questions

Do not use public issues for:

- Security vulnerabilities
- Private profile data
- Private workspace code
- Personal photos or documents
- Secrets or credentials

For security issues, follow `SECURITY.md`.

## Before Opening an Issue

Please check:

- Existing issues
- `README.md`
- `DESIGN.md`
- `IMPLEMENTATION.md`
- `SECURITY.md`
- Recent commits or release notes

## Information to Include

For bugs, include:

- OS and version
- WorkGrid Memory version or commit
- Install method
- Workspace type and approximate size
- Language/framework involved
- Steps to reproduce
- Expected behavior
- Actual behavior
- Logs, if safe to share

Always remove secrets and sensitive profile data before sharing logs.

## Feature Requests

Good feature requests include:

- Problem statement
- Example workflow
- Why it matters
- Possible design
- Security/privacy impact
- Alternatives considered

## Commercial or Private Support

No official commercial support is available yet.

If this changes, details should be added here.
